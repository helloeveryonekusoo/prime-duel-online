# PRIME DUEL

3枚までの数字カードを合計して素数を作り、相手のライフカードを奪う2人用オンライン対戦カードゲームです。ホストが作った6文字のルームコードを使い、別々の端末から参加できます。通信はWebRTCによる端末間の直接接続です。

## 遊び方

- 攻撃側は手札から1〜3枚を選び、選んだカードの数字の合計で素数を作ります。合計値が基本攻撃値になります。
- 7〜13のカードは1枚まで、4〜6のカードも1枚まで選択できます。1〜3には数字帯制限がありません。
- 防御側は手札1〜2枚の数字の合計、または共有の外部ゾーンから防御します。
- 攻撃値と防御値の差以下のライフカードが防御側の手札へ移ります。ライフゾーンが0枚になったら敗北です。
- Aは使用枚数分ドロー、Bは最終攻撃値+1、Cは外部ゾーンの防御値-2です。
- 攻撃はいつでもパスできます。パスしたプレイヤーは次の自分のターン、通常ドロー後に山札の上2枚から1枚を選べます。

## 起動

公開済みのGitHub Pagesを、対戦する両方の端末で開いてください。ローカルで確認する場合は、ES Modulesを読み込める静的HTTPサーバーを使用します。

## ルーム対戦

1. 片方のプレイヤーが名前を入力して「ルームを作る」を押します。
2. 表示された6文字のコードを相手へ伝えます。
3. 相手は名前とコードを入力して「ルームに参加」を押します。
4. 2人がそろうと自動的にゲームが始まります。

## 公開

GitHub Pages上でそのまま動作します。ルームの相手発見には[Trystero](https://trystero.dev/)のMQTT方式を使用し、ゲームデータはWebRTCで端末間を直接送信します。ルームを閉じると対戦状態は破棄されます。

## 構成

- `index.html` - エントリーポイント
- `css/game.css` - レイアウト、カード、レスポンシブ表示
- `js/game.js` - ゲーム状態とルール
- `js/main.js` - 画面描画、操作、WebRTCオンライン同期
