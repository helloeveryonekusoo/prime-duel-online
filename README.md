# PRIME DUEL

数字カードを並べて素数を作り、相手のライフカードを奪う2人用オンライン対戦カードゲームです。ホストが作った6文字のルームコードを使い、別々の端末から参加できます。

## 遊び方

- 攻撃側は手札から1〜3枚を選び、順序を変えて素数を作ります。
- 7〜13のカードは1枚まで、4〜6のカードも1枚まで選択できます。1〜3には数字帯制限がありません。
- 防御側は手札1〜2枚、または共有の外部ゾーンから防御します。
- 攻撃値と防御値の差以下のライフカードが防御側の手札へ移ります。ライフゾーンが0枚になったら敗北です。
- Aは使用枚数分ドロー、Bは最終攻撃値+1、Cは外部ゾーンの防御値-2です。
- 攻撃はいつでもパスできます。パスしたプレイヤーは次の自分のターン、通常ドロー後に山札の上2枚から1枚を選べます。

## ローカル起動

Node.js 20以上をインストールし、次のコマンドを実行します。

```bash
npm install
npm start
```

同じWi-Fi内で遊ぶ場合は、起動したPCで `ipconfig` を実行してIPv4アドレスを確認します。両方の端末から `http://PCのIPv4アドレス:3000` を開いてください。Windowsの確認画面が出た場合は、プライベートネットワークからのアクセスを許可します。

インターネット越しに遊ぶ場合は、WebSocketに対応するサービスへプロジェクト全体を公開してください。

## ルーム対戦

1. 片方のプレイヤーが名前を入力して「ルームを作る」を押します。
2. 表示された6文字のコードを相手へ伝えます。
3. 相手は名前とコードを入力して「ルームに参加」を押します。
4. 2人がそろうと自動的にゲームが始まります。

## 公開

静的ファイルだけではルーム同期ができないため、GitHub Pages単体では動作しません。Render、Railway、Fly.ioなど、Node.jsとWebSocketに対応するサービスを利用してください。開始コマンドは `npm start`、待受ポートは環境変数 `PORT` を自動使用します。

ルームの状態はサーバーのメモリに保持されます。サーバー再起動時にはルームが終了します。

## 構成

- `index.html` - エントリーポイント
- `css/game.css` - レイアウト、カード、レスポンシブ表示
- `js/game.js` - ゲーム状態とルール
- `js/main.js` - 画面描画、操作、オンライン同期
- `server.js` - ルーム作成とWebSocket同期サーバー
- `package.json` - 起動設定と依存パッケージ
